# portfolio-site

A basic React portfolio site built with Vite.

## Features

- Solid color background
- Centered placeholder logo
- Contact section at the bottom (LinkedIn, email)

## Getting Started

```bash
npm install
npm run dev
```

## Customization

- Replace `src/assets/placeholder-logo.svg` with your own logo
- Update the LinkedIn URL in `src/App.jsx` (replace `YOUR_PROFILE`)
- Update the email address in `src/App.jsx` (replace `YOUR_EMAIL@example.com`)
- Adjust the background color in `src/index.css` (`background-color` on `body`)

## Build

```bash
npm run build
```
